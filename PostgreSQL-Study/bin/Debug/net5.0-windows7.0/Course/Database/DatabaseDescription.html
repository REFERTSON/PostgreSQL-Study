Описание таблиц базы данных "Cinema"

1. Таблица Movies (Фильмы)

ID_Movie - Integer (ID фильма)

Title - Character Varying 255 (Название фильма)

Genre - Character Varying 60 (Жанр фильма)

Release_Year - Integer (Год выхода фильма)

Director - Character Varying 100 (Режисёр)

Duration - Integer (Продолжительность; время сколько идет фильм в мин.)

Description - Text (Описание)

2. Таблица Showtimes (Показы)

ID_Showtime - Integer (ID показа)

ID_Movie - Integer (ID фильма)

ID_Hall - Integer (ID зала)

Start_Time - Time without time zone (Время и дата начала показа)

Price - Integer (Цена билета)

3. Таблица Halls (Залы)

ID_Hall - Integer (ID зала)

Capacity - Integer (Кол-во мест в зале)

4. Таблица Tickets (Билеты)

ID_Ticket - Integer (ID билета)

ID_Customer - Integer (ID покупателя)

ID_Showtime - Integer (ID показа)

Seat_Number - Integer (Номер места)

5. Таблица Customers (Покупатели)

ID_Customer - Integer (ID покупателя)

Name - Character Varying 40 (Имя и фамилия покупателя)

Email - Character Varying 100 (Эл. почта)
