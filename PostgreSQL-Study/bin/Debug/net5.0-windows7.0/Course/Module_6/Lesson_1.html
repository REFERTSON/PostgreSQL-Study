Урок по AVG в PostgreSQL

Команда AVG используется для вычисления среднего значения числового столбца. Она полезна для анализа данных, таких как подсчёт среднего рейтинга, цены, года или другого числового показателя.

1. Базовый синтаксис

Синтаксис команды AVG:

SELECT AVG("столбец") FROM имя_схемы."имя_таблицы";

Пример: предположим, у нас есть таблица Showtimes (Покащы). Мы хотим вычислить среднюю цену всех показов:

SELECT AVG("Price") AS "Средняя цена билета" FROM public."Showtimes";

2. Использование AVG с группировкой

Вы можете использовать AVG для расчёта среднего значения по группам данных. Например, вычислим средний год публикации для каждого жанра:


SELECT "Genre", AVG("Price") AS "Средняя цена билета" FROM public."Showtimes" S
JOIN public."Movies" M ON M."ID_Movie" = S."ID_Movie"
GROUP BY "Genre";

3. Особенности использования AVG

  • AVG игнорирует значения NULL в столбце.
  • Для округления результата можно использовать функции, такие как ROUND, но это не рассматривается в этом уроке.

Заключение

Команда AVG предоставляет удобный способ расчёта среднего значения числовых данных в PostgreSQL. Она полезна для анализа данных и вычисления ключевых метрик.
