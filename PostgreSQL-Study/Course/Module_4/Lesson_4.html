Урок по DISTINCT в PostgreSQL

Команда DISTINCT используется для удаления дубликатов из результатов запроса. Она позволяет возвращать только уникальные строки по указанным столбцам.

1. Базовый синтаксис

Синтаксис команды DISTINCT:

SELECT DISTINCT "столбцы" FROM имя_схемы."имя_таблицы";

Пример: найдём уникальные жанры книг из таблицы books:

SELECT DISTINCT "Genre" FROM public."Movies";

3. Использование DISTINCT с несколькими столбцами

Команда DISTINCT может быть использована для возвращения уникальных комбинаций значений нескольких столбцов. Например, уникальные сочетания авторов и жанров:

SELECT DISTINCT author, genre FROM books;

4. Ограничения DISTINCT

  • DISTINCT применяется ко всем указанным столбцам, возвращая уникальные комбинации значений.
  • Для более сложных фильтраций дубликатов можно использовать агрегатные функции, но они не входят в этот урок.

Заключение

Команда DISTINCT позволяет удалять дубликаты из результатов запроса, что делает её полезной для анализа данных и их упрощённого представления. Она легко используется и широко применяется при работе с PostgreSQL.
