Урок по GROUP BY в PostgreSQL

Команда GROUP BY используется для группировки строк, имеющих одинаковые значения в указанных столбцах. Она позволяет выполнять операции агрегации, такие как подсчёт количества, вычисление среднего значения и другие.

1. Базовый синтаксис

Синтаксис команды GROUP BY:

SELECT "столбцы и/или агрегатные_функции" FROM имя_схемы."имя_таблицы" GROUP BY "столбцы";

Пример: подсчитаем количество фильмов в каждом жанре:

SELECT "Genre", COUNT(*) AS "Кол-во фильмов" FROM public."Movies" GROUP BY "Genre";

3. Группировка по нескольким столбцам

Вы можете группировать данные сразу по нескольким столбцам. Например, подсчитаем количество фильмов каждого жанра для каждого года:

SELECT "Release_Year", "Genre", COUNT(*) AS "Кол-во фильмов" FROM public."Movies" GROUP BY "Release_Year", "Genre";

4. Использование GROUP BY с агрегатными функциями

Агрегатные функции, такие как COUNT, SUM, AVG, MIN, MAX, часто используются вместе с GROUP BY. Например, найдём средний год публикации для каждого жанра:

SELECT "Genre", MIN("Release_Year") AS "Минимальный год выпуска" FROM public."Movies" GROUP BY "Genre";

5. Особенности использования GROUP BY

  • Все столбцы, не использующиеся в агрегатных функциях, должны быть указаны в GROUP BY.
  • Выводимые агрегатные данные (например, COUNT, AVG) применяются отдельно к каждой группе.

Заключение

Команда GROUP BY предоставляет мощные возможности для анализа данных, позволяя группировать строки и вычислять агрегированные значения. Она является важной частью языка запросов PostgreSQL.
