Урок по ORDER BY в PostgreSQL

Команда ORDER BY используется для сортировки результатов запроса в PostgreSQL. Вы можете упорядочить строки по одному или нескольким столбцам в порядке возрастания или убывания.

1. Базовый синтаксис

Синтаксис команды ORDER BY:

SELECT "столбцы" FROM имя_схемы."имя_таблицы" ORDER BY столбец [ASC|DESC];

По умолчанию используется сортировка по возрастанию (ASC).

2. Пример: сортировка по году публикации

Рассмотрим пример, в котором книги сортируются по году публикации:

SELECT * FROM public."Movies" ORDER BY "Release_Year";

Этот запрос вернёт список фильмов, упорядоченных по году выпуска от самого раннего к самому позднему.

3. Сортировка по убыванию

Чтобы отсортировать данные в порядке убывания, используйте ключевое слово DESC. Например, фильмы в обратном хронологическом порядке:

SELECT * FROM public."Movies" ORDER BY "Release_Year" DESC;

4. Сортировка по нескольким столбцам

Вы можете упорядочивать результаты по нескольким столбцам. Например, сначала по жанру, а затем по году выпуска:

SELECT "Title", "Genre", "Release_Year" FROM public."Movies" ORDER BY "Genre", "Release_Year";

Этот запрос сначала сортирует фильмы по жанру, а внутри каждого жанра — по году публикации.

Заключение

Команда ORDER BY является мощным инструментом для сортировки данных в запросах PostgreSQL. Она позволяет организовывать результаты так, как это необходимо для анализа или представления данных.
