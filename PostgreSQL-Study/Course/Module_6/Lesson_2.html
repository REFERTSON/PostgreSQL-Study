Урок по SUM в PostgreSQL

Команда SUM используется для вычисления суммы значений числового столбца. Она полезна для подсчёта общих метрик, таких как сумма продаж, общее количество страниц или другие показатели.

1. Базовый синтаксис

Синтаксис команды SUM:

SELECT SUM("столбец") FROM имя_схемы."имя_таблицы";

Пример: предположим, у нас есть таблица Halls (Залы). Мы хотим вычислить общее количество мест в кинотеатре:

SELECT SUM("Capacity") AS "Общее кол-во мест" FROM public."Halls";

2. Особенности использования SUM

  • SUM игнорирует значения NULL в столбце.
  • Для работы с большими таблицами рекомендуется использовать SUM с группировкой (GROUP BY) или фильтрацией (WHERE), чтобы улучшить производительность.

Заключение

Команда SUM предоставляет удобный способ подсчёта общей суммы числовых данных в PostgreSQL. Она широко используется для анализа данных и расчёта ключевых метрик.
