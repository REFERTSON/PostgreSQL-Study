Урок по COUNT в PostgreSQL

Команда COUNT используется для подсчёта количества строк или значений в столбце. Она полезна для анализа данных, например, для подсчёта общего количества записей, уникальных значений или записей с определённым условием.

1. Базовый синтаксис

Синтаксис команды COUNT:

SELECT COUNT(столбец) FROM таблица;

Пример: предположим, у нас есть таблица Movies (Фильмы). Мы хотим узнать общее количество фильмов в таблице:

SELECT COUNT(*) AS "Кол-во фильмов" FROM public."Movies";

2. Использование COUNT для подсчёта уникальных значений

Если вам нужно подсчитать уникальные значения в столбце, используйте COUNT(DISTINCT столбец). Например, подсчитаем количество уникальных жанров:

SELECT COUNT(DISTINCT "Genre") AS "Кол-во жанров" FROM public."Movies";

Заключение

Команда COUNT позволяет легко подсчитывать строки, уникальные значения или записи, соответствующие определённым условиям. Она является важным инструментом для анализа данных в PostgreSQL.
